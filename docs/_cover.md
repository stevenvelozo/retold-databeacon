# Retold DataBeacon <small>0.0.1</small>

> A deployable data beacon for connecting to remote databases, introspecting schemas, generating REST endpoints, and exposing capabilities to the Ultravisor mesh

- Connect to MySQL, PostgreSQL, MSSQL, SQLite and more
- Automatically discover and expose database schemas as REST APIs
- Deploy as a Docker container on any remote machine
- Integrate with Ultravisor for distributed data orchestration
